# Razor_EF
Проект Razor Pages на C#
ASP.net Core 8, EF Core 8

База данных Microsoft SQL Server
Order, Client, Product

При первом старте (пустой или несуществующшей БД) заполняется случайными данными через Bogus (Faker)
С помощью Razor Pages реализованы формы для CRUD - операций для всех таблиц
Использованы Tag-helpers, Html-Helpers, контроллеры, маршруты, BindProperty

Страницы:
Index для выбора таблиц через меню
Страницы для редактирования таблиц (добавление, удаление, изменение)
Для оформления использован Bootstrap

Реализованы:
- валидация ввода данных из модели 
- вывод ошибок через страницу Error (Production) и подробная (Development) 
- вывод подробно протокола работы и ошибок во внешний файл (Serilog)
- REST API (для главной таблицы) с интерфейсом Swagger
- доступ к файлам в физическом каталоге /Files (в комментариях, т.к. не бужет работать в Docker)
- deployment в Dockerfile, docker-compose.yml
